Break

A Break forces Columns inside a Row to split into new rows even if the Columns do not exceed in size.

Code

<Container>
  <Row>
    <Column size="3">Top</Column>
    <Break />
    <Column size="3">Bottom</Column>
  </Row>
</Container>
