Star Wars

We have created a dotnet CLI template for the GraphQL.org Star Wars example. So, you can follow the example and get an impression how you could implement such a schema with our API. In order to get it just run the following dotnet CLI command.

dotnet new -i HotChocolate.Templates.StarWars

After you have installed this template you can just fire up the follwing command in order to get the example up and running:

mkdir starwars
cd starwars
dotnet new starwars
dotnet run --project StarWars/StarWars.csproj -c release

The service should start-up and run on the port 5000. Get GraphiQL and fire up your queries against the schema.
